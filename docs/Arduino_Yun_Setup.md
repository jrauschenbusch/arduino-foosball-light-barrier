# FIRST SETUP
```
1. Install Arduino IDE incl. all drivers (Important!!!)
2. Connect the Arduino Yun with your computer via USB -> Micro USB
3. Start the Arduino IDE
4. Go to Tools (ger. Werkzeuge) and select Board -> Arduino Yun and Port -> COM 14 (Arduino Yun)
5. Select File (ger. Datei) -> Examples (ger. Beispiele) -> Bridge -> YunSerialTerminal
6. Select Sketch -> Upload (ger. Hochladen)
7. Run Tools (ger Werkzeuge) -> Serial Monitor (ger. Serieller Monitor)
8. Select "New Line" (ger. "Neue Zeile") and Press "Send" or hit the enter key
9. Type passwd and hit "send"
10. Set the new password for the root user (Linino Linux)
```
